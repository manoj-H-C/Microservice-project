for auto refresh of microservices when property files are modified in git hub
----
docker pull rabbitmq:4.1.3
docker run --rm -it -p 5672:5672 rabbitmq:4.1.3


for zipkin starting
-----
docker run --rm -it --name zipkin -p 9411:9411 openzipkin/zipkin


for refreshing the property file in one service
POST
http://localhost:8080/actuator/refresh

for refreshing the property file in all services in my app
POST
http://localhost:8080/actuator/busrefresh

for checking health of circuit braker of employee service
GET
http://localhost:8082/actuator/health
